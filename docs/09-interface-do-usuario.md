# Interface do Usuário

## 1. Objetivo

Este documento apresenta as principais telas do SumuBank, o fluxo de navegação entre elas e a organização da interface do usuário.

## 2. Fluxo de Navegação

        Login
        │
        ▼
    Dashboard
        │
    ├──────────────┐
    ▼              ▼
    Contas      Transferências
    │              │
    ▼              ▼
    Extrato      Nova Transferência
    │
    ▼
    Perfil

## 3. Telas

| Tela          | Objetivo                   | Status      |
| ------------- | -------------------------- | ----------- |
| Login         | Autenticação do usuário    | ⏳ Planejada |
| Cadastro      | Cadastro de novos clientes | ⏳ Planejada |
| Dashboard     | Visão geral da conta       | ⏳ Planejada |
| Conta         | Informações da conta       | ⏳ Planejada |
| Extrato       | Histórico de transações    | ⏳ Planejada |
| Transferência | Transferência entre contas | ⏳ Planejada |
| Perfil        | Dados do usuário           | ⏳ Planejada |
| Administração | Gerenciamento do sistema   | ⏳ Planejada |

## 4. Componentes Compartilhados

| Componente       | Utilização             |
| ---------------- | ---------------------- |
| Navbar           | Navegação principal    |
| Sidebar          | Menu lateral           |
| Header           | Informações do usuário |
| Footer           | Rodapé                 |
| Botão Primário   | Ações principais       |
| Campo de Entrada | Formulários            |
| Modal            | Confirmações           |
| Tabela           | Listagem de dados      |
| Card             | Resumo de informações  |
