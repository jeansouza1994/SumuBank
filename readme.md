## Sobre o projeto

O SumuBank é um banco digital fictício desenvolvido para demonstrar a construção de uma aplicação Full Stack moderna utilizando Angular e Spring Boot.

O projeto simula operações bancárias do dia a dia, como abertura de contas, autenticação de usuários, depósitos, saques, transferências e consulta de extrato, aplicando boas práticas de arquitetura, segurança, testes automatizados e infraestrutura.

## Objetivo

Desenvolver um projeto de portfólio que demonstre conhecimentos nas principais tecnologias utilizadas no mercado de desenvolvimento Java, desde a implementação da aplicação até seu deploy em produção.

## Público-alvo

* Recrutadores
* Desenvolvedores
* Líderes técnicos
* Estudantes

## Principais funcionalidades

* Cadastro e autenticação de usuários
* Gerenciamento de contas bancárias
* Depósitos, saques e transferências
* Consulta de saldo e extrato
* Área administrativa para gerenciamento de clientes

## Tecnologias

* Angular
* Java + Spring Boot
* Spring Security
* PostgreSQL
* RabbitMQ
* Docker e Docker Compose
* JUnit e Mockito
* Swagger
* AWS S3
* Prometheus e Grafana

O desenvolvimento será realizado em fases, permitindo a evolução gradual da aplicação:

2. MVP
